📋 安装指南

快速开始，5 分钟完成部署

步骤 1：下载发布包

点击上方"立即下载"按钮，下载完整发布包 ZIP 文件。

步骤 2：解压文件

将下载的 ZIP 文件解压到你选择的目录。

⚠️ 注意：路径不要包含中文或空格

D:\CHXWAI\ 或 /home/user/chxwai/

步骤 3：安装依赖

打开命令行，进入项目目录，安装 Python 依赖。

cd D:\CHXWAI pip install -r requirements.txt

步骤 4：配置 API 密钥

编辑 config/settings.json 文件，填入你的 API 密钥。

{ "api_key": "sk-your-api-key-here", "model": "cloud", "language": "zh-CN" }

步骤 5：启动系统

Windows 用户双击 start.bat，或使用命令行启动。

# Windows start.bat # 或手动启动 python main.py

步骤 6：验证安装

启动后，尝试与 AI 对话，确认系统正常运行。

你：你好 AI: 你好！我是茶海虾王 AI 助手，有什么可以帮助你的吗？

💻 系统要求

确保你的系统满足以下要求

🖥️

操作系统

Windows: 10 或更高版本
macOS: 10.15 或更高版本
Linux: Ubuntu 18.04+ / CentOS 7+

🐍

Python 环境

版本: Python 3.8 或更高
推荐: Python 3.10+

💾

硬件要求

内存: 最低 2GB，推荐 4GB+
存储: 至少 500MB 可用空间

🌐

网络要求

云端模型: 需要互联网连接
本地模型: 可离线运行
