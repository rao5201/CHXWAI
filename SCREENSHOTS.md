# 📸 CHXWAI 项目截图指南

## 需要提供的截图

为了让官网展示更直观的效果，请提供以下截图：

### 1. 命令行界面截图
**内容**：展示 AI 对话的命令行界面
**建议操作**：
```bash
# 启动程序后，进行 2-3 轮对话
你好
帮我查一下天气
```
**截图要求**：
- 包含完整的终端窗口
- 显示至少 2 轮对话
- 终端背景建议使用深色主题
- 分辨率：1200x800 或更高

---

### 2. 网页可视化界面截图
**内容**：展示 Web UI 的主界面
**建议操作**：
```bash
# 启动 Web 服务后访问
http://localhost:5000
```
**截图要求**：
- 包含完整的浏览器窗口
- 显示对话区域和输入框
- 分辨率：1920x1080 或更高

---

### 3. 配置管理界面截图
**内容**：展示配置文件或配置界面
**截图要求**：
- 显示 settings.json 或配置界面
- 展示主要配置项（API 密钥、模型选择等）
- 敏感信息（如 API 密钥）请打码处理

---

### 4. 知识库管理界面截图
**内容**：展示知识库功能
**建议操作**：
```bash
# 演示添加知识和查询知识
记住我喜欢蓝色
我喜欢的颜色是什么？
```
**截图要求**：
- 展示知识添加和查询过程
- 显示知识库文件或管理界面

---

## 截图格式要求

| 项目 | 要求 |
|------|------|
| 格式 | PNG（推荐）或 JPG |
| 尺寸 | 最小 800x600，推荐 1920x1080 |
| 命名 | `screenshot-cli.png`, `screenshot-web.png` 等 |
| 质量 | 清晰，文字可读 |

## 截图替换步骤

1. 将截图文件放入 `CHXWAI/` 目录
2. 重命名为易识别的名称
3. 更新 `index.html` 中的 `<img>` 标签
4. 提交到 GitHub

示例代码：
```html
<div class="screenshot-card">
    <img src="screenshot-cli.png" alt="命令行界面">
    <div class="caption">命令行交互界面 - 简洁高效</div>
</div>
```
