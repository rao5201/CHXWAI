# CHXWAI 博客文章模板

## 新建文章步骤

### 1. 复制模板文件
```bash
cp blog-post-template.html blog-your-article-name.html
```

### 2. 修改文件内容
- 修改 `<title>` 标签
- 修改文章标题、分类、内容
- 修改 meta 描述
- 更新日期和作者

### 3. 更新博客列表页
在 `blog.html` 中添加新文章卡片

### 4. 提交到 GitHub
```bash
git add blog-your-article-name.html
git commit -m "feat(blog): 添加新文章 - 文章标题"
git push
```

## 文章结构建议

### 开头 (Lead)
- 用 1-2 句话概括文章主旨
- 吸引读者继续阅读
- 可包含重要提示或背景信息

### 主体内容
- 使用 H2/H3 标题分段
- 配合代码示例
- 使用列表展示要点
- 插入截图或图表（如有）

### 结尾
- 总结要点
- 提供下一步行动建议
- 添加相关链接

## 文章格式规范

### 代码块
```html
<pre><code># 示例代码
def hello():
    print("Hello, CHXWAI!")</code></pre>
```

### 引用块
```html
<div class="highlight-box">
    <p><strong>提示：</strong> 重要信息放在这里</p>
</div>
```

### 功能卡片
```html
<div class="feature-item">
    <h4>功能名称</h4>
    <p>功能描述</p>
</div>
```

## SEO 优化建议

### Meta 标签
```html
<meta name="description" content="文章简要描述，150-160 字符">
<meta name="keywords" content="关键词 1，关键词 2，关键词 3">
```

### Open Graph
```html
<meta property="og:title" content="文章标题">
<meta property="og:description" content="文章描述">
<meta property="og:type" content="article">
```

### 标题优化
- 包含主要关键词
- 控制在 60 字符以内
- 使用数字和形容词吸引点击
