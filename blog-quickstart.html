5 分钟快速上手茶海虾王

📅 2026-04-08 👤 CHXWAI Team ⏱️ 阅读时间 5 分钟

从零开始，5 分钟完成茶海虾王的安装和配置，快速开始你的 AI 之旅。

💡 提示： 本教程适用于 Windows、macOS 和 Linux 系统，全程无需编程基础。

步骤 1：下载发布包

访问下载页面获取最新版本：

  • 下载中心 →
  • 或直接访问 GitHub Releases

步骤 2：解压文件

将下载的 ZIP 文件解压到合适的目录：

# Windows 示例
D:\CHXWAI\

# macOS/Linux示例
~/chxwai/

⚠️ 注意： 路径不要包含中文或空格，避免潜在问题。

步骤 3：安装依赖

打开命令行工具，进入项目目录，安装 Python 依赖：

cd D:\CHXWAI
pip install -r requirements.txt

步骤 4：配置 API 密钥

编辑 config/settings.json 文件：

{
  "api_key": "sk-your-api-key-here",
  "model": "cloud",
  "language": "zh-CN"
}

🔑 获取 API 密钥： 如果你还没有 API 密钥，可以访问 AI 服务提供商官网获取。

步骤 5：启动系统

Windows 用户：双击 start.bat

macOS/Linux用户：运行 ./start.sh

或手动启动：

python main.py

步骤 6：开始对话

启动后，尝试与 AI 进行第一次对话：

你：你好
AI: 你好！我是茶海虾王 AI 助手，有什么可以帮助你的吗？

🎉 完成！

现在你已经成功安装并运行了茶海虾王！接下来可以：

  • 探索各种功能
  • 配置永久记忆知识库
  • 安装实用插件
  • 自定义你的 AI 助手
